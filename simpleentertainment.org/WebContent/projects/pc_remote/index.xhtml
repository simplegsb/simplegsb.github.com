PC Remote l PC Remote for AndroidTM

Motivation

This project was, strangely enough, a spinoff from the related PC Remote for Android project. During the design of that project it became apparent that the Android specific portion of the project was nothing more than a fancy GUI client for an underlying server-client model. Seeing the possibilities for multiple clients and servers all based off of the same basic model as the PC Remote for Android project, it was decided that it should be a project in its own right. And here we are.

What is it?

PC Remote is a simple client and server that leverages the Java Robot class to provide control of a PC's mouse and keyboard from a remote location.

How do I use it?

To include PC Remote in one of your projects all you need to do is download your favourite flavour and include it in your classpath. You can read the documentation to learn about how to leverage its awesome power. Just remember that we have licensed it using the GPL and as a result you are only able to include it in your project if it, too, is distributed under the GPL.

PC Remote for Android

I wanted an excuse to try my hand at Android development and this was the result. You can use it to control your PC using your Android device.

Source
Repository on GitHub
Download
Android: pcremoteforandroid.apk
Ubuntu: pcremoteserver.deb
Windows: pcremoteserver.zip
Features
  • Control the mouse and keyboard.
  • Automatically detect servers on the network.
  • Switch between multiple servers.
  • Switch between multiple control layouts.
  • Customise and create your own layouts with different controls.
